How it works

If the license is not installed for this, the player will prompt to install. Press the install button and provide the license key. Or alternately you can run the flashLicense.php file that comes with this package as http://domainname/path/flashLicense.php?action=install&prod_id=114. Provide write permission for license.xml file.

If you have bought demo license and you want it to upgrade to the full version, you can repeat the same. Happy capturing :-)
